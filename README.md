# 🔐 GeneretorBCRYPT

Projeto desenvolvido para geração de hashes utilizando o algoritmo BCrypt.

## 🚀 Objetivo

O projeto foi criado com o intuito de facilitar a geração de senhas criptografadas para utilização em sistemas Java Web, APIs e aplicações que necessitam armazenar senhas de forma segura.

---

## 🛠️ Tecnologias Utilizadas

- Java
- JSP / Servlet
- HTML5
- CSS3
- JavaScript
- BCrypt

---

## 🔒 Sobre o BCrypt

O BCrypt é um algoritmo de hash extremamente utilizado para proteção de senhas por possuir:

- Salt automático
- Alta segurança contra ataques de força bruta
- Dificuldade computacional configurável

---

## 📌 Funcionalidades

- Gerar hash BCrypt
- Interface simples e intuitiva
- Compatível com sistemas Java
- Geração instantânea
